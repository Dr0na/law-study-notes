## üìù ANSWER WRITING GUIDELINES (15 MARKS)

### **STRUCTURE:**

**1. Introduction (100-150 words):**
- Context setting
- Define topic
- Mention relevant sections
- Outline structure

**2. Main Content (1,200-1,400 words):**
- Use clear headings and sub-headings
- Explain concepts thoroughly
- Include **minimum 3-4 case laws** (name + principle + year)
- Give illustrations/examples
- Use comparison tables where relevant
- Cite statutory sections
- Include relevant statistics/numbers

**3. Conclusion (100-150 words):**
- Summarize key points
- Modern relevance/significance
- Future trends/reforms if relevant
- Link to constitutional values

---

### **PRESENTATION:**

‚úì **Headings:** Use hierarchical headings (bold, underline)  
‚úì **Paragraphs:** Proper paragraph breaks  
‚úì **Lists:** Use numbered/bulleted lists for clarity  
‚úì **Tables:** Include comparison tables  
‚úì **Citations:** "Case Name (Year)" or "Section X"  
‚úì **Examples:** At least 2-3 illustrations  
‚úì **Underlining:** Underline key terms, case names  
‚úì **Margins:** Leave proper margins  
‚úì **Legibility:** Clear, readable handwriting

## üéì CASE LAW REQUIREMENT

**Minimum Case Laws per 15M Answer:** 3-4

**How to Present:**
- **Full citation:** Case Name v. Case Name (Year) Court Citation
- **Simplified:** Case Name (Year)
- **Principle:** What was held
- **Relevance:** Why relevant to question

**Example:**
**Samar Ghosh v. Jaya Ghosh (2007) 4 SCC 511:** Supreme Court provided illustrative list of acts constituting mental cruelty including false accusations, unreasonable demands, and persistent nagging. Relevant to show mental cruelty includes wide range of conduct.
