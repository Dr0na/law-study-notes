## âœï¸ **How to Write Exam Answers**

### ðŸ“Œ **IDEAL Answer Structure**

```
1. INTRODUCTION (2-3 lines)
   - Define the concept/provision
   - State the Article number
   - Brief constitutional context

2. MAIN BODY (70% of answer)
   - Legal provisions with Article numbers
   - Classification/types (use bullet points)
   - Landmark case laws with year
   - Judicial interpretation
   - Tables/flowcharts if applicable

3. EXCEPTIONS/LIMITATIONS (if applicable)
   - State clearly
   - Cite provisions

4. CONCLUSION (2-3 lines)
   - Current relevance
   - Impact/significance
```

### âœ… **Dos in Exam**

- âœ“ Always cite **Article numbers**
- âœ“ Mention **case names with AIR year** (e.g., Maneka Gandhi AIR 1978)
- âœ“ Use **headings & subheadings**
- âœ“ Draw **tables, flowcharts** wherever possible
- âœ“ Underline **important terms**
- âœ“ Write **legibly** with proper spacing
- âœ“ Use **constitutional language** (e.g., "detenu", "petitioner")

### âŒ **Don'ts in Exam**

- âœ— Don't write long paragraphs
- âœ— Don't forget Article numbers
- âœ— Don't skip case laws
- âœ— Don't write irrelevant content
- âœ— Don't exceed word limit
- âœ— Don't use colloquial language
