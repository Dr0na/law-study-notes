## âœï¸ **HOW TO WRITE ANSWERS IN EXAM**

### **Structure is Key**

**Every Answer Should Have**:
1. **Heading** (question as heading, underlined)
2. **Introduction** (define terms, background) - 10% of answer
3. **Body** (main content with subheadings) - 70% of answer
4. **Criticism/Limitations** (balanced view) - 10% of answer
5. **Conclusion** (summary, relevance) - 10% of answer

---

### **Presentation Tips**

âœ… **DO**:
- Use **headings and subheadings** (makes answer scannable)
- **Underline** key terms, names, sections
- Use **bullet points** for lists
- Draw **diagrams and tables** where relevant
- Write in **points**, not long paragraphs
- Leave space between sections
- **Neat handwriting** (even if small)

âŒ **DON'T**:
- Long, unbroken paragraphs
- No structure or headings
- Forget to define terms
- One-sided (only merits OR only demerits)
- Skip examples
- Poor time management

---

### **Time Management**

**For 80 marks in 180 minutes (3 hours)**:

- Reserve **10-15 minutes** at start to read and plan
- Reserve **15-20 minutes** at end for revision
- Actual writing time: **150 minutes**

**Per Question**:
- **10 marks** = 20-22 minutes (including thinking time)
- **5 marks** = 10-12 minutes
- **2 marks** = 4-5 minutes

**If running out of time**:
- Switch to bullet points
- Complete all questions (something > nothing)
- Partial credit better than zero

---

### **Scoring Tips**

**To Score High**:
1. **Answer all questions** (even partial attempts score)
2. **Stick to word limit** (don't write excessively for any question)
3. **Diagrams add marks** (1-2 marks per good diagram)
4. **Examples matter** (case studies, real-world examples)
5. **Legal provisions** (always mention relevant IPC/CrPC sections)
6. **Balanced analysis** (merits AND demerits, theory AND criticism)
7. **Link concepts** (show integration across modules)
8. **Current relevance** (apply to modern context)

### **Strategy**

**Question Selection**:
- Read ALL questions first (5 minutes)
- Choose questions you know BEST (not first ones you see)
- Plan which to attempt in which order

**Writing Order**:
- Start with **strongest answer** (builds confidence, ensures good start)
- Do high-mark questions first (10-mark, then 5-mark, then 2-mark)
- If time short at end, easier to write short 2-mark answers quickly

- Diagrams worth 1-2 marks each
